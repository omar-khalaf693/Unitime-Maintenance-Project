## Project Description

UniTime is an open-source university timetabling system 
used by universities worldwide. The system helps academic 
institutions manage and optimize their scheduling processes.

### What the system does:

**Course Timetabling:**
- Schedule classes into time slots and rooms
- Assign instructors to courses
- Manage room capacity and availability
- Handle multiple semesters and academic sessions

**Examination Timetabling:**
- Schedule final and midterm exams
- Ensure no student has two exams at the same time
- Optimize exam room assignments

**Student Scheduling:**
- Allow students to submit course requests
- Automatically assign students to class sections
- Detect and resolve scheduling conflicts

**Room Management:**
- Manage all university rooms and their features
- Handle room availability and bookings

**Optimization:**
- Uses a built-in solver to generate optimal schedules
- Supports manual overrides by administrators
- Generates reports in PDF, HTML, and CSV formats
