## Project Description

UniTime is an open-source university timetabling system
used by universities worldwide to manage and optimize
academic scheduling.

### Functional Requirements:
- Create and manage courses
- Assign instructors to courses
- Assign rooms to lectures
- Generate automatic timetables
- Detect scheduling conflicts:
  - Instructor conflicts
  - Room conflicts
  - Time slot conflicts
- Modify existing schedules
- Schedule final and midterm examinations
- Allow students to submit course requests
- Generate reports (PDF, HTML, CSV)
- Role-based access (Admin, Instructor, Student)

### Non-Functional Requirements:
- High performance scheduling algorithm
- Maintainability of code
- Scalability for large systems
- Reliability and correctness
- Security and role-based access control
